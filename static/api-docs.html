API Documentation

Base URL: https://www.imusinfosystems.com/api

Authentication

Admin endpoints require X-Admin-Token header. User endpoints require Authorization: Bearer {token} header.

GET

/operations/social/schedule/draft

Description

Generates a 30-day social media content calendar with AI-themed posts.

Headers

X-Admin-Token: {your_admin_token}

Query Parameters

admin_session (optional): Override token for session-based auth

Response

{ "schedule": [ { "date": "2026-07-04", "time": "10:00", "platform": "linkedin", "theme": "Career Growth", "content": "5 signs you're ready for a promotion..." } ], "total_posts": 30, "status": "draft" }
POST

/operations/social/schedule/publish

Description

Publishes all draft posts to Postproxy for scheduling.

Headers

X-Admin-Token: {your_admin_token} Content-Type: application/json

Body

{ "admin_session": "your_admin_token" }

Response

{ "published": 29, "failed": 1, "total": 30 }
GET

/operations/social/schedule/status

Description

Returns current publishing status across all posts.

Response

{ "scheduled": 15, "published": 8, "failed": 0, "total": 30 }
GET

/operations/referral/stats

Description

Returns referral statistics for the authenticated user.

Headers

Authorization: Bearer {user_session_token}

Response

{ "referrals_pending": [...], "referrals_converted": [...], "total_bonus_xp": 300, "total_credit_inr": 3000 }
GET

/operations/leads/hot

Description

Returns high-scoring leads for priority sales outreach.

Headers

X-Admin-Token: {your_admin_token}

Response

{ "leads": [ { "id": 1, "name": "Rahul Sharma", "email": "rahul@example.com", "score": 87, "status": "hot" } ], "count": 12, "min_score": 70 }
