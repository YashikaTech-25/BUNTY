♿ Accessibility

CareerOS is committed to making job searching accessible for everyone. We follow WCAG 2.1 AA guidelines and continuously improve our accessibility.

Keyboard Navigation

● Compliant

All features are fully accessible via keyboard:

  • Tab navigation through all interactive elements
  • Enter/Space to activate buttons and links
  • Arrow keys for dropdowns and sliders
  • Escape to close modals and dropdowns
  • Skip links to bypass navigation

Screen Reader Support

● Compliant

CareerOS works with all major screen readers:

  • ARIA labels on all interactive elements
  • Semantic HTML5 structure (header, nav, main, footer)
  • Alt text on all images and icons
  • Live regions for dynamic content updates
  • Descriptive link text (no "click here")

Visual Accessibility

● Compliant

Designed for users with visual impairments:

  • High contrast mode (4.5:1 minimum ratio)
  • Dark mode by default (reduces eye strain)
  • Scalable text (up to 200% without breaking layout)
  • Focus indicators visible on all elements
  • No information conveyed by color alone

Motion & Animation

● Compliant

Respects user preferences for motion:

  • Reduced motion support via prefers-reduced-motion
  • No auto-playing animations
  • Optional animations for all transitions
  • No flashing content (no seizures risk)

Known Limitations

○ Partial

We're actively working on these improvements:

  • PDF resume upload accessibility (planned Q3 2026)
  • Video interview prep captions (planned Q3 2026)
  • Voice navigation for mobile (planned Q4 2026)

Testing & Compliance

● Compliant

Our accessibility testing process:

  • Automated testing with axe-core on every build
  • Manual testing with NVDA and VoiceOver
  • User testing with disabled community members
  • Quarterly third-party accessibility audits
  • WCAG 2.1 AA compliance target
